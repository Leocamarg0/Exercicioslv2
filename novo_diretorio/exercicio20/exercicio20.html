A hierarquia obedece as camadas, todivia, id sempre irá prevalecer (em relação às camadas). Quando temos estilos diretamente inseridos dentro do HTML dizemos que esse CSS é "inline". Temos duas variações para o CSS inline, o primeiro caso, é quando criamos uma tag style dentro da tag head do nosso HTML. A segunda maneira de determinarmos um CSS inline é colocando diretamente dentro da tag, da seguinte forma: style "background-color: blue;" Falando sobre os níveis de força do CSS: Em primeiro lugar e mais forte de todos, ou seja, aquele estilos que irá prevalecer em qualquer caso é a utilização da sintaxe !important. O Segundo tipo mais forte é quando determinamos o CSS inline dentro da tag. O terceiro caso mais forte é quando utilizamos o style dentro da tag head do HTML. O quarto e menos forte de todos é quando utilizamos o CSS em uma folha de estilos externa.
